# PresUniv Birthday Board (Batch 2026)

A modern web application built to track and display birthdays for President University Batch 2026 students. This project is developed to provide a clean, accessible, and responsive platform for the student community.

## Features
- **Real-time Birthday Tracking:** View whose birthday is today, tomorrow, or in the upcoming days.
- **Multilingual Support:** Seamless toggling between English (EN) and Indonesian (ID).
- **Dark/Light Theme:** Manual theme switching with local storage persistence.
- **Search & Filtering:** Instantly search for students by name or filter by birth month.
- **Responsive Design:** Fluid grid layouts optimized for mobile, tablet, and desktop views.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Deployment:** Vercel (Recommended)

## Getting Started

### Prerequisites
Make sure you have Node.js installed on your local machine.

### Installation
1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Navigate into the project directory:
   ```bash
   cd presuniv-birthday
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the local development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `app/` - Next.js App Router pages, layout configurations, and global styles.
- `lib/` - Utility functions, types, and static data variables.
